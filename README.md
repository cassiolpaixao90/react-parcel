## Installing

### Steps to setup

**1. Clone the application**

```bash
git clone https://github.com/cassiolpaixao90/react-parcel.git
```

**2. Install dependence the app using NPM or YARN**

```bash
cd react-parcel
npm install
```

**4. Run mode Development**

```bash
npm start
```

**5. Run mode Production**

```bash
npm run build
```
